# FSM状态机 - Idle和Stop状态说明

## 📋 **状态概述**

本FSM状态机实现了两个重要状态：
1. **Idle（待机状态）** - 角色静止不动，等待指令或检测到敌人
2. **Stop（停止状态）** - 停止该角色的所有行动、动画、速度，停止状态机中的所有协程

## 🎯 **Idle状态（待机状态）**

### 功能描述
- 角色完全静止不动
- 停止所有移动和动画
- 检测周围是否有敌人
- 待机一段时间后自动切换到巡逻状态

### 状态行为
1. **进入时**：
   - 停止角色移动（速度设为0）
   - 重置角色旋转角度
   - 重置待机计时器

2. **更新时**：
   - 检测是否有敌人
   - 如果敌人距离在攻击范围内 → 切换到攻击状态
   - 如果敌人距离超出攻击范围 → 切换到追逐状态
   - 如果没有敌人且待机超过2秒 → 切换到巡逻状态

3. **退出时**：
   - 重置待机计时器

### 使用场景
- 角色初始状态
- 战斗结束后的休息状态
- 等待玩家指令的状态

## 🛑 **Stop状态（停止状态）**

### 功能描述
- **完全停止**角色的所有行动、动画、速度
- **停止状态机中的所有协程**
- 角色进入完全静止状态，等待外部指令恢复

### 状态行为
1. **进入时**：
   - 停止所有协程（`StopAllCoroutines()`）
   - 停止物理运动（速度设为0，角速度设为0）
   - 设置为运动学模式，禁用重力
   - 停止动画播放（禁用Animator）
   - 停止行走动画效果
   - 重置角色旋转角度

2. **更新时**：
   - 不执行任何更新逻辑
   - 角色完全静止

3. **退出时**：
   - 恢复物理运动设置
   - 恢复动画播放
   - 恢复行走动画效果

### 使用场景
- 游戏暂停时
- 角色被控制技能影响时
- 过场动画播放时
- 需要完全停止角色行动的特殊情况

## 🎮 **使用方法**

### 方法1：通过FSM_1001直接调用

```csharp
// 获取FSM组件
FSM_1001 fsm = GetComponent<FSM_1001>();

// 停止所有行动
fsm.StopAllActions();

// 恢复行动能力
fsm.ResumeActions();

// 切换到待机状态
fsm.ChangeState(State.Idle);

// 检查是否处于停止状态
bool isStopped = fsm.IsStopped();
```

### 方法2：使用FSMStateExample组件

1. 将`FSMStateExample`脚本添加到角色物体上
2. 运行游戏
3. 使用键盘控制：
   - `S` - 停止角色所有行动
   - `R` - 恢复角色行动能力
   - `I` - 切换到待机状态

### 方法3：在Inspector中测试

在`FSMStateExample`组件上右键可以看到：
- "测试停止角色" - 测试停止功能
- "测试恢复角色" - 测试恢复功能
- "测试待机状态" - 测试待机功能

## 🔧 **技术实现**

### StopState_1001核心功能

```csharp
public void OnEnter()
{
    // 1. 停止所有协程
    fsm.StopAllCoroutines();
    
    // 2. 停止物理运动
    rb.velocity = Vector2.zero;
    rb.angularVelocity = 0f;
    rb.isKinematic = true;
    rb.gravityScale = 0f;
    
    // 3. 停止动画
    animator.enabled = false;
    
    // 4. 停止行走动画
    StopWalkBob();
}
```

### IdleState_1001核心功能

```csharp
public void OnUpdate()
{
    // 检测敌人并决定下一步行动
    if (fsm.currentEnemy != null)
    {
        float distance = Vector2.Distance(fsm.transform.position, fsm.currentEnemy.position);
        
        if (distance <= fsm.AttackRange)
        {
            fsm.ChangeState(State.Attack);
        }
        else
        {
            fsm.ChangeState(State.Chase);
        }
    }
    else if (idleTimer >= IDLE_DURATION)
    {
        fsm.ChangeState(State.Patrol);
    }
}
```

## ⚠️ **注意事项**

1. **Stop状态优先级**：Stop状态会完全停止角色，包括所有协程
2. **状态恢复**：从Stop状态恢复时会恢复到Idle状态
3. **物理设置**：Stop状态会修改Rigidbody2D的设置，退出时会恢复
4. **动画控制**：Stop状态会禁用Animator组件
5. **协程管理**：Stop状态会停止所有协程，包括攻击协程
